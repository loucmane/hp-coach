## Gap architecture (5 gaps: 3 collocation, 1 connective, 1 polarity)
Meets blueprint minimum (≥1 collocation, ≥1 polarity, ≥1 connective) and the batch brief's emphasis on **collocation-misfit + polarity-mirror**.

| gap | type | key | trap set (all real words) |
|---|---|---|---|
| 1 | collocation | circulation | rotation / suspension / distribution — real -tion/-sion nouns that collocate with *other* frames ("in rotation", "in suspension"); none takes "keep the value in ___" |
| 2 | collocation | absorb | cushion (a blow) / alleviate (pain) / dampen (enthusiasm) — verbs whose real objects are not "a bill/cost" |
| 3 | connective | However | Moreover (addition) / Thereafter (temporal) / Namely (specification) — the frame turns *against* the upbeat paragraph, so only the concessive fits |
| 4 | polarity | fragile | agile (positive **polarity mirror**) / mercantile (thematic sense-misfit) / futile (negative sense-misfit) — suffix-rhymed -ile set |
| 5 | collocation | cracks | fissures / seams / fractures — gap/split nouns; only "cracks" carries "slip through the cracks" |

**Polarity gap (4) is deliberately not form-eliminable.** The four -ile options are not "3 positive + 1 negative": *agile* is the positive mirror, but *futile* is also negative and *mercantile* is polarity-neutral/thematic. A skimmer cannot pick the key by "the only negative word" — they must read that the concessive frame ("than the applause admits") plus the downstream fragility evidence (thin seam of high-value goods; two platforms slipping into the red once grants expire; ledger "reads more soberly") demands *unstable*, which is *fragile*, not *doomed* (*futile*) or *commercial* (*mercantile*).

**One spelling variety**: BrE only.

## Self-blind-solve (from passage alone, arguing each option)
- Gap 1 → **circulation**. "keep the value in circulation" is the money idiom; rotation/suspension/distribution have no such collocation with a stored value. Single fit.
- Gap 2 → **absorb**. "absorb a repair bill/cost"; cushion/alleviate/dampen collocate only with blow/pain/enthusiasm. Single fit.
- Gap 3 → **However**. Paragraph pivots from upbeat to caution; addition/temporal/specification adverbs all misfire. Single fit.
- Gap 4 → **fragile**. Concessive frame + fragility evidence forces the negative-yet-precise word; agile (wrong polarity), futile (wrong sense/too strong), mercantile (wrong sense). Single fit.
- Gap 5 → **cracks**. "slip through the cracks (of the statistics)"; fissures/seams/fractures lack the idiom. Single fit.
