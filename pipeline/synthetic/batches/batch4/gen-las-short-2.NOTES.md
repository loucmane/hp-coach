# gen-las-short-2 — authoring notes

## Genre / topic rationale
- **Topic:** language & media — dialect leveling among young people and their national-streaming media diet. Sits in this batch's Unit-2 pool (dialect change + public-service/broadcast media + reading habits). No overlap or neighbouring with any batch 1–3 topic.
- **Genre:** sakprosa / debatt_opinion (essä), framing-claim opening (`Med jämna mellanrum återkommer larmet`). First-person voice, normative modality (`bör vi bry oss mindre om…`), explicit stance, byline close on a fictional columnist. The opinion register gives genuine authorial stance for the hållning item and a clean home for the stance-inversion trap.
- **Anti-tidiness:** the essay concedes real leveling (`Jag förnekar inte att en utjämning pågår`) rather than staging a frictionless win — the concession is load-bearing for Q2's overgeneralisation distractor.

## Planted trap architecture
**Q1 — enligt_texten_detalj (key C).** Planted target (p2): `använde det lokala tjocka l:et mer sällan — men bara just det draget`, with peers outweighing screen time. Hedged, scoped to one pronunciation feature, directional (media → feature).
- A = **overgeneralisation** — `hela deras dialekt ... ord, uttal och satsbyggnad`; contradicted by `Deras ord och satsbyggnad bar dialekten kvar`.
- B = **reversed_causality** — arrow flipped (lost feature → more media).
- D = **scope_shift** — moves the result from pronunciation to vocabulary, the exact level the text says held.

**Q2 — forfattarens_hallning (key B).** Stance target: `den är en svag kraft bredvid umgänget` + `att göra medievanorna till den stora boven är att förväxla det som syns med det som verkar`.
- A = **stance_inversion** (NOT reversed_causality) — media as `den främsta orsaken`, the alarm she rebuts.
- C = **overgeneralisation** (opposite pole) — `helt opåverkade ... oförändrade`, against her explicit concession.
- D = **plausible_worldknowledge** — depopulation/urbanization leveling; a common belief the text never argues.

## Stem-law compliance
Q1 stem asks what the study found about the students' language — the outcome could have been "everything eroded", "nothing changed", or "one feature only"; the stem does not entail the key. Q2 stem asks which stance the writer takes among several the essay could plausibly hold.

## Distractor-form-law compliance
Every distractor is refutable only from passage content, not by form. Q2 is deliberately form-symmetric: key B and stance-inversion A both read as confident claims (they tie as the longest options), so neither "pick the hedged one" nor "pick the longest" works. The key is the longest option in Q1 only, breaking the tell across the unit.

## Self-blind-solve
Solved both from the passage alone. Q1 = C uniquely: the vocab/syntax-held sentence kills A and D, and the media→feature direction kills B. Q2 = B uniquely: A is the rebutted alarm, C contradicts the concession, D imports an unstated depopulation premise. Key letters spread C, B (across both batch-4 units: B, C, C, B — no all-A column). All entities fictional (Tobias Hedman, Marika Öberg). Passes M-SCHEMA / M-BANDS / M-TELL / M-PLAGIARISM.
