# NOTES — gen-las-short-1 (hemberedskap / household crisis preparedness)

## Topic & genre rationale
- **Topic:** *hemberedskap* — household crisis preparedness — from my exclusive pool.
  The specific angle is deliberately narrow: the argument that the state's stockpiling
  message neglects **collective/neighbourhood** preparedness (*grannsamverkan*). This is
  distinct from every batch 1–5 topic and is not a neighbour of any used item.
- **Genre:** sakprosa / **debatt_opinion**, `framing-claim` opening. Register inherits from
  genre: explicit stance, normative modality (*borde*, *måste*-adjacent *inte kan beordra*),
  first-person voice, nominalisation (*beredskapen*, *samordning*, *överenskommelse*),
  `-s`-passive-adjacent constructions. Sentence length varies (short verdicts —
  "Det ligger något i det." — beside longer subordinated sentences).
- **Frame:** title line (not repeated in passage); fictional byline last
  (Gunnel Åkerlund, tidigare krisberedskapssamordnare). No glossary — no specialist term
  needs one (would trip Batch 1's glossary-defines-absent-word catch otherwise).
- **Fiction / anti-plagiarism:** the answerable content is an **invented** survey
  (Institutet för samhällsberedskap i Härnösand, autumn power outage). The real Swedish
  "en vecka på egen hand" campaign is background only, never the answer key — a
  knowledgeable solver cannot answer Q1 without reading the invented finding (>3 dygn,
  flerbostadshus). Härnösand is a real city, the institute is fictional (same convention as
  the reference unit's "Institutionen för landskapsekologi i Vadstena").

## Planted trap architecture

### Q1 — `enligt_texten_detalj` (key A)
Planted target (para 2): a **hedged, directional, scoped** finding — households that
**coordinated in advance** coped best (not those with the biggest stockpiles), effect
appearing **only after outages >3 days** and **clearest in apartment blocks**.
- **B = reversed_causality**, deliberately **hedged in form** (no absolutizer): flips the
  arrow — big stockpiles *caused* coordination to emerge. Keeps the key from being the sole
  measured option (M-FORM) and breaks "pick the qualified answer" (Law 10).
- **C = overgeneralisation** — "oavsett hur långt avbrottet blev" absolutises the scoped
  ">3 dygn" result and drags the cause back to stockpiles.
- **D = scope_shift** — attributes the effect to *villaområden*, where the text says the
  difference was small (effect was clearest in *flerbostadshus*): wrong housing, wrong cause.
- Every distractor is a plausible-but-wrong *mechanism* a passage-blind reader cannot
  dismiss (stockpile-enabled mutual aid; careful planning; wealthier detached-house areas).

### Q2 — `forfattarens_hallning` (key B)
Whole-text stance: own preparedness is wise **but insufficient**; the message *should*
elevate stairwell/neighbourhood/village alongside the backpack — spans lede + finding +
explicit *borde*-claim.
- **A = detail_as_main** — weaponises the concession ("samordning kan man inte beordra fram")
  into the main thesis, adding "det enda som ytterst håller"; the author says the opposite
  (the stockpile is *insufficient*). Not verbatim-true (Law 11): the "det enda"-overstatement
  is the identifiable flaw.
- **C = reversed_causality** (stance inversion) — "förråd blir onödiga"; the text states
  flatly that an agreement "ersätter aldrig ett förråd", so stockpiles are **not** unnecessary.
- **D = overgeneralisation / plausible_worldknowledge** — a state takeover the author never
  argues for; "enskilda klarar inte" is contradicted by the many households that coped.
